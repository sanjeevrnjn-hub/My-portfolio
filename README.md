# 🌐 Sanjeev Ranjan — Personal Portfolio Website

A modern, responsive full stack portfolio website built from scratch.

🔗 **Live Demo:** Coming Soon

## 🛠️ Technologies Used

### Frontend

- **HTML5** — Structure and content
- **CSS3** — Styling, animations, glassmorphism effects
- **JavaScript (Vanilla)** — Interactivity, typing animation, scroll effects
- **Google Fonts (Roboto)** — Typography
- **Font Awesome** — Icons

### Backend

- **Node.js** — JavaScript runtime for server
- **Express.js** — Web framework for building REST API
- **Mongoose** — MongoDB object modeling for Node.js

### Database

- **MongoDB** — NoSQL database for storing contact form messages

### Tools & Others

- **Git & GitHub** — Version control and code hosting
- **VS Code** — Code editor
- **Live Server** — Local development server
- **Nodemon** — Auto-restarts backend server on file changes
- **dotenv** — Manages environment variables securely

## 🚀 How to Run Locally

### Prerequisites

- Node.js installed
- MongoDB installed locally

### Steps

**1. Clone the repository:**

```bash
git clone https://github.com/sanjeevrnjn-hub/My-portfolio.git
```

**2. Open frontend:**

- Open `index.html` with Live Server in VS Code

**3. Start the backend:**

```bash
cd backend
npm install
npm run dev
```

**4. Create .env file inside backend folder:**

```
MONGO_URI=mongodb://localhost:27017/portfolio
PORT=5000
```

**5. Visit:**

```
http://127.0.0.1:5500/index.html
```
